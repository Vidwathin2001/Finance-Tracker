# Cloud-Based Finance Tracker Web Application

A full-stack finance tracking web application deployed on AWS EC2 using Node.js, Express, MongoDB Atlas, HTML, CSS, and JavaScript.

## 🚀 Features

- User Registration and Login
- Add Income and Expense Transactions
- View Financial Records
- MongoDB Atlas Database Integration
- AWS EC2 Cloud Deployment
- Responsive User Interface

---

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Cloud Platform
- AWS EC2

---

## 📂 Project Structure

```bash
finance-tracker/
│
├── backend/
│   ├── config/
│   ├── models/
│   ├── routes/
│   ├── server.js
│   ├── package.json
│   └── .env
│
├── frontend/
│   ├── index.html
│   ├── dashboard.html
│   ├── style.css
│   └── script.js
│
└── README.md
```

---

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/finance-tracker.git
```

### 2. Navigate to Backend

```bash
cd backend
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Create `.env` File

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

### 5. Start Server

```bash
node server.js
```
